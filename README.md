# Telegram Card Validator Bot

A professional Python-based Telegram bot that validates credit card numbers using the Luhn Algorithm and forwards valid hits to a specific channel.

## Setup
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`.
3. Create a `.env` file with `BOT_TOKEN` and `CHANNEL_ID`.
4. Run with `python main.py`.
